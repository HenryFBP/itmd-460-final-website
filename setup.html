Pre-requisites

  • Human requirements

    • Use a mouse, keyboard, or other human interface device
    • Navigate your filesystem
    • Navigate websites
    • Click on dialogue boxes
    • Understand and navigate user interfaces on your screen
    • Perform respiration
    • Be conscious

  • Hardware requirements

    1. You need a laptop, preferably with 4 Gigabytes or more of RAM.

      It can be running Windows, OSX, or Linux.

    2. You need one two things to run the apps you'll make:
      • An Android phone, or
      • Virtualization and VT-x features to be enabled on your processor. These are features that not all processors have. I will go over how to check that later.

Once you've got your laptop in front of you, you're ready for the next step.

Installing

  1. Navigate to https://developer.android.com/studio/ using your web browser.

  2. Click on "Download Android Studio".

  3. Accept the license agreement.

  4. Execute the file that you just downloaded.

    Don't forget to sharpen your guillotine for this part.

  5. Click on 'Next' for all the prompts.

    Now we wait.

  6. Now that it's done installing, you can click 'Finish'.

    This will open Android Studio's setup wizard.

  7. If you're asked to import settings, just select 'Do not import settings' and click OK.

  8. The setup wizard will ask you how you want to configure settings. This is up to you, but you can use 'Standard'.

  9. More downloading! (This step is likely downloading the source code of libraries you will use to develop apps with so that proper syntax highlighting and IntelliSense can work, among other things.)

    This step is also installing an android emulator and Intel HAXM, a virtualization engine that makes emulating Android devices much faster.

  10. You're done with the setup steps!

    Before you is the 'Welcome Screen'.
